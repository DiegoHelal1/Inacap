Bienvenidos a mi sitio web

Este es un sitio web creado por un alumno del ramo de "programacion en front end" con la intencion de que sea evaluada por el profesor, para saber si cumple con las capacidades de uso de "HTML y CSS" y si sabe usar los recursos para poder crear una pagina web.

Por mi parte eh aprendido de que alfinal depende de uno y las intenciones que tenga para dejar bien creada una pagina web tanto estructuralmente hablando como el modelado.

Me eh dado cuenta a lo largo de la creacion esta pagina que uno no puede saber todas las funciones existentes para el modelado de una caja ("div, /div") y las paginas que recomiendo para ver distintos modelados puedes usar tanto "www.w3wchool.com" o "www.getboostrap.com".

Es una increible actividad para saber que alfinal uno se pone limitaciones en la creacion de estos proyectos, por lo tanto, tener en cuenta de que uno se pones sus propias trabas y que no existe limitaciones para dejar una pagina web en perfectas condiciones y una buena visualizaci√≥n para el expectador.
